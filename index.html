Lägg till spel

Det här kopplar portalen till ditt GitHub-repo, så att spel du laddar upp här läggs till direkt i repot och dyker upp för alla som besöker sidan (efter att GitHub Pages byggt om, oftast under en minut).

Token sparas bara lokalt i den här webbläsaren. Dela den aldrig med någon. Skapa helst en "fine-grained" token med behörighet begränsad till bara det här repot (Contents: Read and write) — se README för instruktioner.
